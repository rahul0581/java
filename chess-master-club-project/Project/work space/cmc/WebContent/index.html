Welcome to Chess Masters Club!

Chess master club is a cross-platform application that allows Two players to engage in a real time game of chess. The primary Goal is to provide chess game that is intuitive and entertaining for The player of all skill levels. The game also provides some useful Features such as the capability to pause a program and save it and Return to it later. In general it focuses on providing a simple stream Line playing.
